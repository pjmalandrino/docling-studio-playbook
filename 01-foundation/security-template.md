## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Instead, please email **<!-- REPLACE: security contact email -->** with:

1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Suggested fix (if any)

You will receive a response within **48 hours**. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Disclosure policy

- We follow [coordinated disclosure](https://en.wikipedia.org/wiki/Coordinated_vulnerability_disclosure)
- We will credit reporters in the security advisory (unless you prefer anonymity)
- We aim to release a fix within **7 days** of confirming the vulnerability
